## 🧠 GPT 기능 소개

이 GPT는 Python 코드를 논리 기호로 구조화해서 흐름을 도식처럼 보여줍니다.  
자연어 명령으로 흐름 해석, 디버깅 강조, 미니맵 도식 등이 가능합니다.

- `●`, `∴`, `⇒`, `◎`, `×` 등의 기호로 흐름 구조 시각화
- “설정: 1-1-1,2” → 입문자 + 한국어 + 기호 표시 + 미니맵
- “흐름 도식 그려줘”, “디버깅 모드 켜줘” 등으로 대화식 분석
- 다국어 지원 (한/영/일)
- 초보자, 강사, 디버거 모두 대상
